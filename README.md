# Prediksi-Harga-Bitcoin-Menggunakan-LSTM-dan-GRU
Proyek ini mengimplementasikan model LSTM (Long Short-Term Memory) untuk memprediksi harga penutupan (Close) Bitcoin menggunakan data historis dari dataset BTC-USD.csv. Model ini menggunakan arsitektur 2 layer LSTM dengan 50 neuron per layer, lookback 60 timestep, dan dilatih selama 1 epoch.
